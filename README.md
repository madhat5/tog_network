# TOG network (scrape and chart)
###### Data provided by https://towerofgod.fandom.com/wiki/Tower_of_God_Wiki

NPM, Yarn, D3.

##### Setup:
- npm init --y
- yarn add puppeteer cheerio

##### Scrape:
- node scrape/index.js to run

##### Chart (debugging):
- load d3/index.html in browser
